## Overview

**Big O notation** is a mathematical notation used to describe the upper bound of an algorithm's time or space complexity. It helps developers understand how an algorithm's performance scales as the input size grows, which is crucial for writing efficient code.

<Info>
Common Big O complexities from fastest to slowest: O(1) constant, O(log n) logarithmic, O(n) linear, O(n log n) linearithmic, O(n²) quadratic, O(2ⁿ) exponential.
</Info>
