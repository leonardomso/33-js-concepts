## Overview

ES6 introduced four new collection types: **Map**, **Set**, **WeakMap**, and **WeakSet**. These provide alternatives to plain objects and arrays with specific use cases — Maps for key-value pairs with any key type, Sets for unique values, and Weak variants for memory-efficient object references.

<Info>
**What you'll learn in this guide:**
- Map vs Object — when to use each
- Set vs Array — unique value collections
- WeakMap and WeakSet — garbage collection friendly references
- Real-world use cases for each collection type
- Performance characteristics and best practices
</Info>
