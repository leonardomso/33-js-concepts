## Overview

JavaScript uses prototypal inheritance, where objects can inherit directly from other objects. Every object has an internal link to another object called its prototype. That prototype object has a prototype of its own, and so on until an object is reached with `null` as its prototype. This chain of prototypes is called the **prototype chain**.
