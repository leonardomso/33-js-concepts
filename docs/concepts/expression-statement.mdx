## Overview

According to the ECMAScript specification, **expressions** produce a value, and **statements** are instructions to perform an action, such as variable assignment or control flow.

<Info>
Function declarations are hoisted and can be called before they are defined in the code, while function expressions are not hoisted and must be defined before being invoked.
</Info>
